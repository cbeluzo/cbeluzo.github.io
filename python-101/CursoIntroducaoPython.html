3. Estruturas Condicionais e de Repetição

Neste tópico, vamos aprender sobre as estruturas condicionais e de repetição em Python, que são fundamentais para controlar o fluxo de execução de um programa e para automatizar tarefas repetitivas.

Utilização do if, else e elif para Controle de Fluxo

As estruturas condicionais if, else e elif permitem que você tome decisões com base em condições específicas. Vejamos como elas funcionam em Python:

# Exemplo de estrutura condicional if-else
idade = 18
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")
# Exemplo de estrutura condicional if-elif-else
nota = 75
if nota >= 90:
    print("Aprovado com A")
elif nota >= 80:
    print("Aprovado com B")
elif nota >= 70:
    print("Aprovado com C")
else:
    print("Reprovado")

Abordagem dos Laços de Repetição com for e while

Os laços de repetição for e while permitem que você execute um bloco de código repetidamente. Vejamos como utilizá-los:

# Exemplo de laço de repetição for
for i in range(5):
    print(i)
# Exemplo de laço de repetição while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

Exemplificação por Meio de Exemplos Práticos

A melhor maneira de entender estruturas condicionais e de repetição é através de exemplos práticos. Vamos ver alguns exemplos que combinam essas estruturas:

  • Exemplo de Verificação de Número Par ou Ímpar:

    numero = 7
if numero % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")

  • Exemplo de Laço de Repetição com Lista:

    numeros = [1, 2, 3, 4, 5]
for num in numeros:
    print(num)

  • Exemplo de Utilização do break em um Laço de Repetição:

    contador = 0
while True:
    print(contador)
    contador += 1
    if contador >= 5:
        break

  • Exemplo de Utilização do continue em um Laço de Repetição:

    for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

4. Funções

Neste tópico, vamos explorar as funções em Python, que são blocos de código reutilizáveis projetados para realizar uma tarefa específica. Vamos entender como definir, chamar e trabalhar com funções.

Definição e Chamada de Funções

As funções em Python nos permitem encapsular blocos de código para que possam ser reutilizados facilmente. Vejamos como definir e chamar uma função:

# Exemplo de definição de função

    print("Olá, mundo!")

# Chamada da função

Explanação sobre Parâmetros, Argumentos e Retorno de Valores

As funções podem receber dados chamados de parâmetros e retornar resultados. Vamos entender como funcionam os parâmetros, argumentos e retorno de valores em Python:

# Exemplo de função com parâmetros e retorno de valor
def somar(a, b):
    return a + b

# Chamada da função com argumentos
resultado = somar(3, 5)
print(resultado)

Discussão sobre o Escopo de Variáveis e Funções Aninhadas

O escopo em Python refere-se à acessibilidade de variáveis ​​e funções em diferentes partes do código. Vamos discutir sobre o escopo de variáveis e funções aninhadas em Python:

# Exemplo de escopo de variáveis
x = 10  # Variável global

def minha_funcao():
    y = 5  # Variável local
    print(x + y)  # Variável global pode ser acessada dentro da função

minha_funcao()

# Exemplo de função aninhada
def externa():
    x = 10

    def interna():
        return x * 2

    return interna()

print(externa())

5. Coleções em Python

Neste tópico, vamos explorar as coleções em Python, que são estruturas de dados que permitem armazenar múltiplos itens em uma única variável. Vamos entender como funcionam as listas, tuplas e dicionários, bem como as operações comuns e técnicas avançadas de manipulação de coleções.

As listas, tuplas e dicionários são tipos de coleções em Python, cada uma com suas próprias características e métodos de acesso aos elementos."

  • Listas: Mutáveis e ordenadas, representadas por colchetes [].
  • Tuplas: Imutáveis e ordenadas, representadas por parênteses ().
  • Dicionários: Mutáveis e desordenados, representados por chaves {} e pares de chave-valor.

  1. Considerações sobre Tuplas

    Inserção

    As tuplas são imutáveis, então não é possível adicionar ou remover elementos depois que são criadas. No entanto, é possível criar uma nova tupla com os elementos desejados.

    Ordenação

    Tuplas são imutáveis, portanto, não podem ser ordenadas diretamente. Você pode converter a tupla em uma lista, ordenar a lista e depois converter de volta para uma tupla, se necessário.

    Remoção

    Não é possível remover elementos de uma tupla depois que ela é criada, pois tuplas são imutáveis.

  1. Ordenação em Dicionários

    Considere o seguinte dicionário em Python:

# Criando um dicionário vazio
meu_dicionario = {}

# Inserindo elementos no dicionário
meu_dicionario['chave1'] = 'valor1'
meu_dicionario['chave2'] = 'valor2'

    Ordenação por chave

    Quando ordenamos um dicionário por chave, estamos reorganizando seus elementos com base nas chaves, em ordem crescente ou decrescente, conforme desejado.

# Ordenando o dicionário pelas chaves em ordem crescente
dicionario_ordenado_chave = dict(sorted(meu_dicionario.items()))

# Ordenando o dicionário pelas chaves em ordem decrescente
dicionario_ordenado_chave_decrescente = dict(sorted(meu_dicionario.items(), reverse=True))

    No código acima, sorted() é uma função que ordena os itens do dicionário com base nas chaves. A função items() retorna uma lista de tuplas onde cada tupla contém uma chave e seu respectivo valor. Em seguida, sorted() classifica essas tuplas com base nas chaves. Finalmente, dict() converte a lista ordenada de tuplas de volta em um dicionário.

    Ordenação por Valor

    Quando ordenamos um dicionário por valor, estamos reorganizando seus elementos com base nos valores associados às chaves, em ordem crescente ou decrescente, conforme desejado.

# Ordenando o dicionário pelos valores em ordem crescente
dicionario_ordenado_valor = dict(sorted(meu_dicionario.items(), key=lambda item: item[1]))

# Ordenando o dicionário pelos valores em ordem decrescente
dicionario_ordenado_valor_decrescente = dict(sorted(meu_dicionario.items(), key=lambda item: item[1], reverse=True))

     Neste caso, sorted() é uma função que ordena os itens do dicionário com base nos valores. O parâmetro key=lambda item: item[1] especifica que a ordenação deve ser realizada com base no segundo elemento de cada tupla (ou seja, nos valores). Novamente, dict() converte a lista ordenada de tuplas de volta em um dicionário.

    Remoção de elementos de um dicionário

   # Removendo um elemento com uma chave específica
   del meu_dicionario['chave1']

   # Removendo e obtendo um elemento com uma chave específica
   valor_removido = meu_dicionario.pop('chave2')

6.1 Introdução à Biblioteca Pandas

Neste tópico, vamos explorar a biblioteca Pandas, uma poderosa ferramenta para manipulação e análise de dados em Python. Vamos aprender sobre as operações básicas e principais funcionalidades oferecidas pelo Pandas.

Instalação e Importação da Biblioteca Pandas

Antes de começarmos, é importante garantir que a biblioteca Pandas esteja instalada em seu ambiente Python. Você pode instalá-la usando pip:

pip install pandas

Em seguida, vamos importar a biblioteca Pandas em nosso ambiente de trabalho:

import pandas as pd

Criação de DataFrames

O DataFrame é a estrutura de dados central do Pandas, que nos permite representar e manipular conjuntos de dados tabulares. Vamos aprender a criar DataFrames a partir de diferentes fontes de dados:

# Criando um DataFrame a partir de um dicionário
dados = {'Nome': ['Alice', 'Bob', 'Charlie'],
         'Idade': [25, 30, 35]}
df = pd.DataFrame(dados)

Leitura e Escrita de Dados

O Pandas oferece suporte para ler e escrever dados de e para vários formatos, como CSV, Excel, SQL, JSON, entre outros. Vamos aprender como fazer isso:

# Leitura de um arquivo CSV para um DataFrame
df = pd.read_csv('dados.csv')

# Escrita de um DataFrame para um arquivo CSV
df.to_csv('novo_dados.csv', index=False)

Exploração de Dados

Uma vez que tenhamos nossos dados carregados em um DataFrame, podemos realizar uma variedade de operações para explorar e entender nossos dados melhor:

# Visualizando as primeiras linhas do DataFrame
df.head()

# Obtendo informações sobre o DataFrame
df.info()

# Resumo estatístico do DataFrame
df.describe()

Seleção e Fatiamento de Dados

Podemos selecionar e fatiar dados de um DataFrame de várias maneiras para realizar análises específicas:

# Selecionando uma coluna específica
coluna_idade = df['Idade']

# Fatiando linhas e colunas
subconjunto = df.loc[1:3, ['Nome', 'Idade']]

Manipulação de Dados

O Pandas oferece uma variedade de métodos para manipular e transformar os dados em um DataFrame conforme necessário:

# Adicionando uma nova coluna
df['Sexo'] = ['F', 'M', 'M']

# Renomeando colunas
df.rename(columns={'Idade': 'Anos'}, inplace=True)

# Preenchendo valores ausentes
df.fillna(0, inplace=True)

Visualização de Dados

Por fim, podemos visualizar nossos dados de maneira clara e informativa usando as capacidades de visualização do Pandas e outras bibliotecas, como Matplotlib e Seaborn:

# Gráfico de barras da contagem de valores em uma coluna
df['Sexo'].value_counts().plot(kind='bar')

# Gráfico de dispersão entre duas variáveis
df.plot.scatter(x='Idade', y='Salário')

Operações Básicas com Arrays

Podemos realizar uma variedade de operações básicas com arrays NumPy, como aritmética, manipulação de forma e indexação:

# Operações aritméticas entre arrays
resultado = arr * 2

# Manipulação de forma de arrays
arr_reshape = arr.reshape(1, 5)

# Indexação de arrays
elemento = arr[0]

Funções Matemáticas e Estatísticas

O NumPy oferece uma ampla gama de funções matemáticas e estatísticas para trabalhar com arrays NumPy:

# Funções matemáticas
media = np.mean(arr)
maximo = np.max(arr)
raiz_quadrada = np.sqrt(arr)

# Funções estatísticas
mediana = np.median(arr)
desvio_padrao = np.std(arr)

Operações de Álgebra Linear

O NumPy possui suporte abrangente para operações de álgebra linear, como multiplicação de matriz e decomposição de matriz:

# Multiplicação de matriz
matriz1 = np.array([[1, 2], [3, 4]])
matriz2 = np.array([[5, 6], [7, 8]])
produto = np.dot(matriz1, matriz2)

# Decomposição de matriz
autovalores, autovetores = np.linalg.eig(matriz)

Indexação Avançada e Fatiamento

Podemos realizar indexação avançada e fatiamento de arrays NumPy para acessar e manipular partes específicas dos dados:

# Indexação avançada
elementos_indices = arr[[0, 2, 4]]

# Fatiamento
sub_array = arr[1:4]

Funções do NumPy úteis para análise de dados e computação científica

  1. np.arange(): Cria um array com uma sequência de números.

    arr = np.arange(0, 10, 2)  # Cria um array de 0 a 10 (exclusive) com passo 2

  2. np.linspace(): Cria um array com uma sequência de números espaçados de forma uniforme.

    arr = np.linspace(0, 10, 5)  # Cria um array de 0 a 10 com 5 elementos igualmente espaçados

  3. np.zeros(): Cria um array preenchido com zeros.

    arr = np.zeros((3, 3))  # Cria uma matriz 3x3 preenchida com zeros

  4. np.ones(): Cria um array preenchido com uns.

    arr = np.ones((2, 2))  # Cria uma matriz 2x2 preenchida com uns

  5. np.eye(): Cria uma matriz identidade.

    matriz_identidade = np.eye(3)  # Cria uma matriz identidade 3x3

  6. np.concatenate(): Concatena arrays ao longo de um eixo especificado.

    arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])
concatenado = np.concatenate((arr1, arr2))  # Concatena os arrays

  7. np.unique(): Retorna os valores únicos em um array.

    valores_unicos = np.unique(arr)  # Retorna os valores únicos no array

  8. np.where(): Retorna os índices onde uma condição é verdadeira.

    indices = np.where(arr > 5)  # Retorna os índices onde os valores são maiores que 5

  9. np.random.randint(): Gera números inteiros aleatórios dentro de um intervalo especificado.

    numeros_aleatorios = np.random.randint(0, 10, size=5)  # Gera 5 números inteiros aleatórios entre 0 e 10

  10. np.random.normal(): Gera números aleatórios de acordo com uma distribuição normal.

    dados_normal = np.random.normal(loc=0, scale=1, size=(3, 3))  # Gera uma matriz 3x3 de números aleatórios com média 0 e desvio padrão 1

6.3 Introdudção a biblioteca Seaborn

Neste tópico, vamos explorar a biblioteca Seaborn, uma poderosa ferramenta de visualização de dados estatísticos em Python. Vamos aprender sobre as operações básicas e principais funcionalidades oferecidas pelo Seaborn.

Instalação e Importação da Biblioteca Seaborn

Antes de começarmos, é importante garantir que a biblioteca Seaborn esteja instalada em seu ambiente Python. Você pode instalá-la usando pip:

pip install seaborn

Em seguida, vamos importar a biblioteca Seaborn em nosso ambiente de trabalho:

import seaborn as sns
import matplotlib.pyplot as plt  # Importando também o Matplotlib para personalização adicional

Carregamento de Dados de Exemplo

O Seaborn oferece conjuntos de dados de exemplo para praticar visualização. Vamos carregar um desses conjuntos de dados para uso nos exemplos a seguir:

# Carregando o conjunto de dados 'iris' (dados de flores)
dados_iris = sns.load_dataset('iris')

Visualizações Básicas com Seaborn

Podemos criar uma variedade de visualizações básicas com o Seaborn para explorar os dados e entender as relações entre as variáveis:

# Gráfico de dispersão para visualizar a relação entre duas variáveis
sns.scatterplot(data=dados_iris, x='sepal_length', y='sepal_width')

# Histograma para visualizar a distribuição de uma variável
sns.histplot(data=dados_iris, x='petal_length', kde=True)

# Gráfico de barras para comparar categorias
sns.countplot(data=dados_iris, x='species')

Visualizações Avançadas com Seaborn

Além das visualizações básicas, o Seaborn oferece uma variedade de gráficos avançados para análise exploratória de dados, incluindo gráficos de distribuição, gráficos de categorias, gráficos de relacionamento e gráficos de matriz:

# Gráfico de caixa para comparar distribuições entre diferentes categorias
sns.boxplot(data=dados_iris, x='species', y='petal_length')

# Gráfico de violino para comparar distribuições e densidades
sns.violinplot(data=dados_iris, x='species', y='sepal_length')

# Gráfico de matriz para visualizar relações entre múltiplas variáveis
sns.pairplot(data=dados_iris, hue='species')

Personalização de Visualizações

O Seaborn oferece muitas opções de personalização para ajustar a aparência das visualizações de acordo com suas necessidades:

# Personalizando o estilo dos gráficos
sns.set_style('whitegrid')

# Adicionando título e rótulos aos eixos
plt.title('Relação entre Comprimento e Largura das Sépalas')
plt.xlabel('Comprimento da Sépala')
plt.ylabel('Largura da Sépala')

# Exibindo a legenda
plt.legend()

# Exibindo o gráfico
plt.show()
